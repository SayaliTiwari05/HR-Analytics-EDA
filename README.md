# HR Analytics EDA

## Project Description
Performed Exploratory Data Analysis on HR dataset to identify factors affecting employee attrition.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Key Steps
- Data Cleaning and Preprocessing
- Handling Missing Values
- Feature Analysis
- Data Visualization

## Insights
- Employees with lower salary have higher attrition
- Work-life balance affects retention
- Certain job roles show higher turnover
